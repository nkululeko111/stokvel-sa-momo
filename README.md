# MoMo Stokvel+

MoMo Stokvel+ is a comprehensive digital financial inclusion platform built on **MTN Mobile Money (MoMo) APIs**.  
It transforms South Africa's traditional stokvel culture by enabling communities to **save, invest, and build wealth together** while promoting financial literacy and inclusion.


## 🌍 Problem
Traditional stokvels serve millions of South Africans but face significant challenges:
- **Manual processes** leading to errors and disputes
- **Cash-based transactions** creating security risks
- **Limited financial education** restricting growth potential
- **Language barriers** excluding non-English speakers
- **Lack of transparency** causing trust issues
- **No digital integration** with modern payment systems


## 💡 Solution
MoMo Stokvel+ provides a complete digital transformation:

### 🏦 **Digital Stokvel Management**
- **MTN MoMo Integration**: Seamless mobile money transactions
- **Automated Contributions**: Set-and-forget recurring payments
- **Multi-language Support**: English, isiZulu, isiXhosa, Afrikaans
- **Real-time Transparency**: Live balance tracking and member activity

### 📚 **Financial Education Hub**
- **Interactive Learning Modules**: Stokvel basics, digital payments, wealth building
- **Financial Calculators**: Savings projections, stokvel returns
- **Local Language Content**: Education in native languages
- **Achievement System**: Gamified learning experience

### 📊 **Advanced Analytics**
- **Personal Financial Insights**: Track savings growth and patterns
- **Stokvel Performance**: Monitor group health and projections
- **Trust Scoring**: Build financial reputation
- **Goal Tracking**: Set and achieve financial milestones

## 🚀 Getting Started

### Prerequisites
- **Node.js 18+**
- **PostgreSQL 14+**
- **Redis 6+**
- **MTN MoMo Developer Account**
- **Expo CLI** (for mobile development)

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Configure your environment variables
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Environment Configuration
Set up your `.env` file with:
- MTN MoMo API credentials
- Database connection strings
- JWT secrets
- Redis configuration
